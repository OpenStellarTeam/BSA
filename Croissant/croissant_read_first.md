# 匿名评审说明 / For Anonymous Review

为了确保论文匿名评审的顺利进行，我们已将 Croissant 数据集的在线链接替换为本地路径。

To ensure the anonymity of the paper review process, we have replaced the online URL links with local paths for the Croissant dataset.

## 数据使用说明 / Data Usage Instructions

1. 请下载 `croissant.json` 和 `data` 文件夹，并将它们放在同一目录下
   Please download `croissant.json` and the `data` folder, and place them in the same directory

2. 或者，您也可以直接使用 `data/dataset.csv` 文件
   Alternatively, you can directly use the `data/dataset.csv` file
